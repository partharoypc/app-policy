## Project Information

A web app that is free, open-source, and perfect for generating a generic or GDPR-compliant privacy policy and terms of use for your mobile apps. 📄✨

### 🛡 2026 Professional Edition
This version has been upgraded to meet the highest industry standards for **2026 compliance**, including:
- Mandatory **Account & Data Deletion** paths.
- Comprehensive **Data Safety** transparency.
- **AI Integration** disclosures.
- **Children's Privacy** (COPPA/GDPR-K).

### 🛠 Technology Stack
- **HTML5 & Vanilla Javascript**: Pure, fast, and dependency-free core logic.
- **Bootstrap 5.3 & Premium CSS**: A compact, minimal, and professional design system.
- **Lucide Icons**: Minimalist, professional iconography.
- **Simple Icons**: Monochrome SVG brand logos.
- **Glassmorphism**: A modern UI with subtle blur effects and a deep slate palette.

### 🚀 Getting Started
1. Open `index.html` in any modern web browser.
2. Complete the streamlined **7-step wizard**.
3. Generate and export your documents in **HTML** or **Markdown**.

### 📁 Structure
- `index.html`: Main application entry point.
- `css/premium.css`: Professional minimalist styling.
- `js/app.js`: Core application engine.
- `js/thirdpartyservices.min.js`: Monochrome SVG logo data.
